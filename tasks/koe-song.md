## パロディアプローチのメモ

### 替え歌のコンセプト

原曲「声 / KOE」は**失われた人の声を探す**歌。
替え歌「声デバイス」は**声を失わないための機械を作る**歌。

方向が逆なだけで、核心にある動機は同じ:
「大切な音が消えてしまうことへの恐怖」

### 音節の対応

| 原曲 | 替え歌 | 音節数 |
|------|--------|-------|
| こうさてん の ノイズを すりぬけて | きばん の うらがわで ハンダが さめて | 12 |
| ずっと きみの りんかくを さがしていた | またも よあけに バイナリを やいた | 13 |
| なみがたに へんかんされた その こえの さきへ | ESP32に ながしこんだ いのりの さきへ | 14 |
| おとのない せかいで ぼくだけが しっている | おとのない しんやで ぼくだけが しんじてる | 14 |

### ガジェット→感情変換表

| Koe Deviceの技術用語 | 感情的な意味 |
|---------------------|------------|
| リングバッファ | 記憶が上書きされる前に |
| 5秒前から録音 | 奇跡はいつも予告なく始まる |
| Morning Mix | 昨夜の君を朝に聴く |
| GPS時刻同期 | 空から降ってくる共通の「今」 |
| 0.3ミリ秒以下の同期 | 言葉より精密な繋がり |
| COINデバイス | 胸ポケットに入る楽器 |
| ループが始まった | 録音が音楽になった瞬間 |

### Weird Al meets Shibuya-kei

Weird Alのアプローチ: 技術用語を素直に歌詞に放り込む  
Shibuya-keiのアプローチ: でもメロディーと感情は一切妥協しない  
結果: 「ESP32」という単語が普通に泣ける、という倒錯した美しさ
