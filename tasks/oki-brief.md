ハワイの家に **Koe Stone** を 12 個置かせてもらってます。

### 使い方 (Tech 好き Oki さん向け詳細)

| アクション | 結果 |
|----------|------|
| 持ち上げる | 加速度センサーで wake、LED 1回呼吸 |
| 上面に触れる | 音楽再生 (事前キュー済み、1秒で鳴る) |
| 指2本で触れる | 音量 low/mid/high 3段サイクル |
| 逆さまにする | Mute (ハードストップ) |
| 別の Stone に触れる | 再生引き継ぎ (移動可) |
| 置く | 鳴り続ける |
| 上面5秒長押し | 英語音声ガイド "Hi. Touch me to play music." |
| NFC タップ | `koe.live/start` が開く |
| 誰も触らず30分 | フェードアウト、スリープ |

### 充電
- 底面 **Qi 5W** 内蔵、iPhone 用の普通の充電パッド(同梱3枚)に置くだけ
- バッテリー: 再生 **8時間**, 待機 **30日**
- 全12台を輪番で充電すれば滞在期間持ちます
- 充電状態は LED 色で: 青=満, 緑=使用中, 橙=残20%, 赤=残5%

### ネットワーク
- **WiFi不要**。全 Stone は出荷時ペアリング済み、BLE mesh 自動構築
- Oki さんの iPhone から AirPlay 2 で音源をキャスト可能 (「Koe Stone」と出る)
- ネット無しでも内蔵プリロード 10曲 (Hawaii vibes インスト) で自律再生

## Admin (Oki さん専用)

### 管理ダッシュボード
URL: **https://koe.live/admin/hawaii** (1日1回アクセス推奨)
パスワード: Yuki から別途 Signal/LINE で送付

表示内容:
- 12台の heartbeat 状況 (5分毎)
- 各台のバッテリー残量 / 再生時間 / 最後に触られた時刻
- 故障検知, 温度警告, 再起動ボタン
- OTA firmware update (Yuki から push されるのを approve するだけ)

### 遠隔デバッグ
もし何か動作がおかしい場合:
1. その台を Oki さんが **一回逆さまにする** (リセットパルス)
2. 5秒待って起こす (再起動)
3. それでもダメなら dashboard から "Reboot" ボタン
4. それでもダメなら Yuki に Signal で "Stone 003 dead"

### FW 更新
もし Yuki が滞在中に FW を更新したい場合 (バグ修正等):
1. Oki さん dashboard に update 通知が出る
2. "Approve" ボタンで全 12 台が 10分以内に自動更新
3. 音楽は途切れません (A/B partition + rollback)

## Troubleshooting

| 症状 | 対処 |
|------|------|
| 音が出ない | 逆さま→正位置で再起動 |
| 同期ズレ | 一番近くの Stone を5秒長押し (mesh再構築) |
| バッテリー減り早い | dashboard で該当台の health 確認 |
| LED 消灯 | Qi パッドに置く (深放電からの復帰) |
| 全台反応なし | 屋内 Wi-Fi router 近くに1台置いて router 経由で mesh 再起動 |
