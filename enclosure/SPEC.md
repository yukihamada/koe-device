# Koe Device Enclosure Specification

## Dimensions
- External: 38 x 42 x 13mm
- Wall thickness: 1.2mm
- Corner radius: 3mm
- Material: Resin (SLA) or PLA (FDM prototype)

## Internal Layout (top to bottom)
```
┌──────────────────────┐
│  ███ Speaker 15x10   │  ← top, sound holes
│  ○ LED (window)      │
│  ─── PCB 30x35mm ── │  ← middle
│  ● ● MIC holes x2   │  ← bottom face, angled 120 deg
│  ████ Battery ██████ │  ← bottom
│  [USB-C]  [Button]   │  ← side
└──────────────────────┘
```

## Features
- Top: Speaker grille (0.5mm holes, 3x5 grid)
- Top: LED light pipe / translucent window (2mm diameter)
- Bottom: 2x microphone ports (1mm holes with acoustic mesh)
- Side (right): USB-C cutout (9.0 x 3.3mm)
- Side (right): Button access hole (2mm diameter)
- Back: Clip mount point (lanyard hole or magnetic mount)

## Assembly
- 2-piece snap-fit (top shell + bottom shell)
- 2x M1.4 self-tapping screws for secure closure
- Battery held by friction fit + foam tape
- PCB support standoffs (4x, 1.5mm height)

## Wearable Options
1. **Pendant**: Lanyard hole → neck
2. **Clip**: Spring clip mount → shirt pocket
3. **Magnetic**: 8mm neodymium magnet recess → attach to clothing
