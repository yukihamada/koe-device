# DK Edition ケース 3Dプリント発注

## JLCPCB 3Dプリント
1. https://jlcpcb.com/3d-printing にアクセス
2. `hardware/cases/dk-case.stl` をアップロード
3. 素材: SLA 8001 Resin (Black)
4. 数量: 10個
5. 見積もり: ~$2-3/個 = ~$30

## 代替: PCBWay 3Dプリント
1. https://www.pcbway.com/rapid-prototyping/3d-printing/ にアクセス
2. 同じSTLをアップロード
3. 素材: SLA Resin (Black)
4. 数量・見積もりを比較

## STLファイル
- パス: `hardware/cases/dk-case.stl`
- サイズ: 135.6 x 65.6 x 24.5mm
- 3Dプレビュー: https://koe.live/3d
