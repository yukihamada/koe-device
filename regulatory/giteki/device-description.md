| 項目 | 内容 |
|------|------|
| 製品名 | Koe（声）AIボイスコンパニオン |
| 型番 | KOE-PICK-001 |
| 製造者 | EnablerDAO |
| 連絡先 | info@enablerdao.com |
| 製品カテゴリ | 小電力データ通信システム / 特定無線設備 |

## 2. 製品の機能概要

Koe は、音声認識とAI応答機能を備えた小型ウェアラブルデバイスです。WiFi経由でクラウドサーバーと通信し、ユーザーの音声入力に対してAIによる応答を提供します。BLE（Bluetooth Low Energy）はスマートフォンとのペアリング・初期設定に使用します。

## 8. モジュール実装条件

本製品におけるESP32-S3-MINI-1-N8R2の実装は、以下の条件を満たしています:

1. **アンテナクリアランス:** モジュールのアンテナ部分（PCBアンテナ）周辺に、Espressif社推奨の最低限のクリアランス（15mm以上）を確保
2. **GNDプレーン:** モジュール直下に十分なGNDプレーンを配置
3. **電源デカップリング:** モジュールの電源ピン近傍に100nF + 10uFのバイパスコンデンサを配置
4. **RF特性への影響:** 金属部品をアンテナ近傍に配置していない。筐体はRF透過性の樹脂素材を使用
