## 4. 技術基準への適合性

- 証明規則 別表第1号 1(1)の2

### 4.2 認証済みモジュールの使用に関する確認

本製品は、工事設計認証（認証番号: 201-220017）を取得済みの無線モジュール ESP32-S3-MINI-1-N8R2 を搭載しています。以下の条件を満たすことを確認しました:

#### (a) モジュールの使用条件の遵守

- [x] モジュールのデータシートに記載された電気的仕様範囲内で使用している
- [x] 推奨動作電圧範囲（3.0V〜3.6V）内で給電している
- [x] モジュールのファームウェアによるRF出力制御を変更していない
- [x] 認証時の送信出力上限を超えていない（WiFi: 20dBm以下、BLE: 10dBm以下）

#### (b) アンテナに関する確認

- [x] モジュール内蔵PCBアンテナをそのまま使用（外部アンテナなし）
- [x] アンテナ周辺に推奨クリアランス（15mm以上）を確保
- [x] アンテナ近傍に金属部品を配置していない
- [x] 筐体材質はRF透過性素材（ABS樹脂/シリコン）を使用

#### (c) 実装に関する確認

- [x] Espressif社「Hardware Design Guidelines」に準拠したPCBレイアウト
- [x] モジュール直下に十分なGNDプレーンを確保
- [x] 電源デカップリングコンデンサを適切に配置
- [x] モジュールのRFポートに追加回路を接続していない
