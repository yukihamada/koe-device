# 技適（技術基準適合証明）取得ガイド — Koe デバイス

## 技適とは

技適（技術基準適合証明 / Technical Conformity）は、日本国内で無線機器を使用・販売するために必要な認証制度です。電波法に基づき、総務省が定める技術基準に適合していることを証明する必要があります。

技適マークのない無線機器を日本国内で使用すると、電波法違反（1年以下の懲役または100万円以下の罰金）となります。

## 本デバイスの状況

### 重要: モジュール認証済み

Koe デバイスが搭載する **ESP32-S3-MINI-1-N8R2**（Espressif Systems製）は、既に技適認証を取得済みです。

| 項目 | 内容 |
|------|------|
| モジュール名 | ESP32-S3-MINI-1-N8R2 |
| 工事設計認証番号 | 201-220017 |
| 認証取得者 | Espressif Systems (Shanghai) Co., Ltd. |
| 対応規格 | WiFi 802.11 b/g/n (2.4GHz) + Bluetooth LE 5.0 |

モジュールが認証済みのため、**最終製品での認証プロセスは大幅に簡略化**されます。ただし、以下を証明する書類の提出が必要です:

- モジュールをデータシート通りの条件で使用していること
- 製品の筐体・アンテナ配置がRF性能を劣化させていないこと
- モジュールの認証条件（アンテナ利得、送信出力等）を逸脱していないこと

## 認証の2つのパス

### パス1: 工事設計認証（Full Certification）

登録証明機関（TELEC等）による正式な認証。量産・一般販売向け。

| 項目 | 詳細 |
|------|------|
| 対象 | 量産品（数量制限なし） |
| 期間 | 2〜4週間（モジュール認証済みの場合） |
| 費用 | 30万〜80万円（テスト内容による） |
| メリット | 販売数量に制限なし、正式な技適マーク付与 |
| デメリット | 費用が高い、設計変更時に再認証が必要な場合あり |

### パス2: 技術基準適合自己確認（Self-Declaration）

2014年の電波法改正で導入された制度。少量生産・試験販売向け。

| 項目 | 詳細 |
|------|------|
| 対象 | 少量生産（目安: 100台以下） |
| 期間 | 1〜2週間（テスト結果があれば） |
| 費用 | 10万〜30万円（最低限のテスト費用） |
| メリット | 費用が安い、手続きが簡単 |
| デメリット | 届出が必要、記録の保管義務（10年間） |

### 推奨パス

初回ロット（〜100台）は **パス2（自己確認）** で市場投入し、量産時に **パス1（工事設計認証）** へ移行することを推奨します。

## タイムライン

```
Week 1-2:  書類準備・テスト計画策定
Week 2-3:  テストラボでの測定（予約状況による）
Week 3-4:  結果確認・書類提出
Week 4-5:  認証取得 / 自己確認届出完了
```

モジュール認証済みのため、フル認証でも通常の半分程度の期間で完了する見込みです。
