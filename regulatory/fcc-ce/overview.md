# FCC / CE 認証概要 — Koe デバイス

> **優先順位:** まず日本市場向けに技適を取得し、その後FCC（米国）/ CE（EU）を取得する。
> 技適テストの測定データは FCC/CE 申請にも一部流用可能。

---

## 1. FCC認証（米国市場）

### 1.1 適用規格

| 項目 | 内容 |
|------|------|
| 規制 | FCC Part 15, Subpart C, Section 15.247 |
| 対象 | 2.4GHz帯スペクトラム拡散・デジタル変調機器 |
| 認証種別 | Certification（認証） |
| 管轄 | Federal Communications Commission (FCC) |

### 1.2 モジュール認証済みの利点

ESP32-S3-MINI-1-N8R2 は FCC認証済み（FCC ID確認要）です。認証済みモジュールを搭載する最終製品は、以下の条件を満たすことで **Certification を省略し、DoC（適合宣言）で対応可能** な場合があります:

- モジュールを無改造で使用
- 内蔵アンテナをそのまま使用
- モジュールのFCC Grant条件を逸脱しない

ただし、**意図的放射体（Intentional Radiator）** を含むため、最終的な認証要否は FCC認定テストラボとの確認が必要です。

### 1.3 必要な試験（最終製品）

| 試験項目 | 内容 | 省略可否 |
|----------|------|---------|
| Conducted Emissions | 伝導放射（電源ライン） | 必須 |
| Radiated Emissions | 放射エミッション（筐体） | 必須 |
| Band Edge Compliance | 帯域端の放射レベル | モジュール認証でカバー可能 |
| Peak / Average Power | 最大/平均出力 | モジュール認証でカバー可能 |
| Antenna Gain | アンテナ利得確認 | 内蔵アンテナなら省略可 |

### 1.4 費用・期間の目安

| 項目 | 目安 |
|------|------|
| テスト費用 | $3,000〜$8,000（約45万〜120万円） |
| FCC申請費用 | $1,000〜$2,000 |
| 期間 | 4〜8週間 |
| FCC ID表示 | 製品ラベルまたは電子表示 |

### 1.5 FCC IDの表示

モジュールのFCC IDを最終製品に表示する場合（モジュール認証を利用）:

```
Contains FCC ID: [モジュールのFCC ID]
This device complies with Part 15 of the FCC Rules.
```

## 2. CE認証（EU市場）

### 2.1 適用指令

| 指令 | 内容 |
|------|------|
| RED (2014/53/EU) | Radio Equipment Directive（無線機器指令） |
| LVD (2014/35/EU) | Low Voltage Directive（バッテリー駆動のため適用外の場合あり） |
| RoHS (2011/65/EU) | 有害物質の使用制限 |
| WEEE (2012/19/EU) | 廃電気電子機器指令 |

### 2.2 適用する整合規格

| 規格 | 内容 |
|------|------|
| EN 300 328 | 2.4GHz帯広帯域伝送システム（WiFi/BLE） |
| EN 301 489-1 | EMC共通要件 |
| EN 301 489-17 | EMC（2.4GHz帯機器固有要件） |
| EN 62368-1 | 安全性（Audio/Video/IT機器） |
| EN 62479 | 人体SAR/EMF評価（低電力機器の簡易評価） |

### 2.3 モジュール認証済みの利点

ESP32-S3-MINI-1-N8R2 は CE認証済みです。最終製品では以下の確認が必要です:

- **EMC試験:** 筐体込みでの放射エミッション・イミュニティ試験は必要
- **RF試験:** モジュール認証データの流用が可能（最終製品での確認測定は推奨）
- **安全性試験:** バッテリー搭載のため、EN 62368-1 の関連部分の評価が必要
- **SAR/EMF評価:** 20dBm以下・身体近接使用の場合、EN 62479 による簡易評価で対応可能

### 2.4 必要な書類

| 書類 | 内容 |
|------|------|
| EU適合宣言書 (DoC) | 製造者による適合宣言 |
| Technical File | 技術文書一式 |
| テストレポート | 認定試験所(NB)の試験報告 |
| ユーザーマニュアル | EU公用語（最低英語）での取扱説明書 |
| ラベル・マーキング | CEマーク + 情報表示 |

### 2.5 費用・期間の目安

| 項目 | 目安 |
|------|------|
| EMC試験 | EUR 3,000〜6,000（約50万〜100万円） |
| RF試験（確認測定） | EUR 2,000〜4,000（約35万〜65万円） |
| 安全性評価 | EUR 1,500〜3,000（約25万〜50万円） |
| 技術文書作成支援 | EUR 1,000〜2,000（約15万〜30万円） |
| **合計** | **EUR 7,500〜15,000（約125万〜245万円）** |
| 期間 | 6〜12週間 |

### 2.6 CEマーキング

```

```

製品本体またはパッケージに表示。最小サイズ 5mm。

## 3. 技適との相乗効果

### 3.1 テストデータの共有

技適の測定で取得したデータは、FCC/CE申請に一部流用できます:

| 技適テスト項目 | FCC流用 | CE流用 |
|---------------|---------|--------|
| 不要発射（スプリアス） | 部分的に可 | 部分的に可 |
| 空中線電力 | 可 | 可 |
| 占有周波数帯幅 | 可 | 可 |
| 周波数偏差 | 可 | 可 |

**推奨:** 技適テスト実施時に、FCC/CEで必要な追加測定（伝導エミッション、イミュニティ等）も同時に行うことで、総費用を削減できます。

### 3.2 推奨スケジュール

```
Phase 1 (Month 1-2):   技適取得 → 日本市場投入
Phase 2 (Month 3-5):   FCC取得 → 米国市場投入
Phase 3 (Month 4-6):   CE取得  → EU市場投入
```

FCC と CE は並行して進めることが可能です。

## 4. 推奨認証パートナー

グローバル認証を一括で対応できるパートナーを推奨します:

| 機関 | 技適 | FCC | CE | 備考 |
|------|------|-----|-----|------|
| UL Japan | o | o | o | 全認証ワンストップ対応 |
| TUV Rheinland Japan | o | o | o | CE に特に強い |
| SGS Japan | o | o | o | グローバル最大ネットワーク |
| Bureau Veritas | - | o | o | 海外認証に強い |
| TELEC | o | - | - | 技適特化、国内最安 |

**推奨:** 技適は TELEC（コスト最適）で取得し、FCC/CE は UL Japan または TUV Rheinland で一括取得するのが効率的です。
