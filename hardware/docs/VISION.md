# Koe — プロダクトビジョン

## 一言で

**群衆を楽器にするデバイス。**

## なぜ他と違うのか

### 他のAIウェアラブルが失敗した理由

| 製品 | 提案 | 失敗の理由 |
|------|------|-----------|
| Humane AI Pin | スマホの代替 | スマホの方が圧倒的に便利 |
| Rabbit R1 | AIエージェント | スマホアプリで十分 |
| Friend pendant | 常時録音AI | プライバシー懸念、何が嬉しいか不明 |

共通の問題: **「スマホで既にできること」を別デバイスでやろうとしている。**

### Koeが違う理由: スマホにできないことをやる

スマホにできないこと:
1. **100人が同時に同じ音を同期再生する** — Bluetoothでは不可能
2. **群衆の中で、自分の近くで音を聞く** — PAスピーカーから100m離れたら音は0.3秒遅れる
3. **全員の声を拾って1つの音楽にする** — 物理的にスマホのマイクでは無理(ノイズだらけ)
4. **周りの人と音で繋がる** — AirDropですら手動。Koeは電源を入れるだけ

## 3つの革命的体験

### 1. Crowd Orchestra (群衆オーケストラ)

**フェスの観客全員がハミングすると、AIがリアルタイムでハーモニーを生成する。**

- 観客Aが「ド」をハミング → デバイスがピッチ検出
- 観客Bが「ミ」、Cが「ソ」 → システムがコード(Cmaj)を認識
- AIが足りないパートを生成 → 全デバイスから補完音が流れる
- 1000人が楽器を弾けなくても、全員で音楽が作れる

**これは人類史上初めて可能になる体験。** 技術的には:
- 各デバイスでピッチ検出(ESP32のDSPで可能)
- UDPマルチキャストでピッチデータを共有(音声データより100x軽い)
- STAGEのPi CM5でハーモニー計算 + 配信
- 全デバイスで同期再生(GPS時刻同期)

### 2. Sound Memory (音の記憶)

**「さっき誰かが言ってた店の名前なんだっけ？」→ デバイスに聞くと、その瞬間の音声が再生される。**

- 常時録音(ローカルリングバッファ30秒 + クラウドに圧縮保存)
- セマンティック音声検索:「東京」と言った瞬間を検索
- オーディオブックマーク: ダブルタップで「この瞬間を覚えて」
- デイリーダイジェスト: AIが1日を2分に要約して再生

競合との違い: Friend pendantも録音するが「テキスト」に変換して終わり。
Koeは**音声そのものを記憶として扱う**。声のトーン、背景音、感情 — テキストでは失われる情報を保持する。

### 3. Spatial Crowd (空間群衆)

**フェス会場の全デバイスのマイクが、リアルタイムで3D音声マップを作る。**

- 1000台のデバイスが各々の位置(GPS)と周囲の音(マイク)をSTAGEに送信
- STAGEが全データを合成 → 会場全体の3Dオーディオマップを生成
- アーティストは「群衆のどこで最も盛り上がっているか」を音でリアルタイムで知れる
- 録音として残せば「あのフェスの体験」を後から3Dで再生できる

## フォームファクターの再定義

### 「テクノロジーを感じさせない」デザイン

ジョブズの原則: **テクノロジーは消えるべき。体験だけが残る。**

- **画面ゼロ、ボタン1つ** — 操作を覚える必要がない
- **音で全てを伝える** — ビープ音でモード、状態、バッテリーを知らせる
- **温度フィードバック** — 録音中は微かに暖かく、待機中はクール(ペルチェ素子)
- **呼吸ハプティクス** — AIが考えている時、赤ちゃんの呼吸のように微かに振動する
- **経年変化する素材** — 真鍮フレームが使い込むほど色が変わる(パティーナ)。あなただけのKoe

### 充電の儀式

充電器は「石の台座」。Koeを台座に置くと、まるで川の石を元の場所に戻すような感覚。
磁石でカチッとハマる。LEDが月のように満ちていく。

## ポジショニング

### 今のポジション(弱い)
「小さいAIデバイス」 → Friend/Humane/Rabbitと同じカテゴリ

### 新しいポジション(唯一無二)
「人と人を音で繋ぐデバイス」 → 新しいカテゴリを作る

| 既存カテゴリ | Koeのカテゴリ |
|------------|-------------|
| スマートスピーカー(1人→1台) | 分散音響(N人→N台→1つの音) |
| AIアシスタント(AI→人) | 音の記憶(環境→AI→人) |
| PA/音響設備(プロ専用) | 群衆楽器(全員参加) |
