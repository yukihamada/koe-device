# Koe Device — 3つのフォームファクター

## 共通回路: 全モデル同一PCB or 派生

---

## 1. Koe MINI — 超小型ペンダント/クリップ

```
    ╭──────────╮
    │  ◎ mic   │  ← 25 x 30 x 9mm
    │ [ESP32]  │     (AirTag以下)
    │  ● LED   │
    │  ─ USB-C │
    ╰──────────╯
      ↕ 約5g

  正面          側面
  ┌────┐       ┌──┐
  │○  ●│       │▒▒│ ← 9mm薄
  │    │       │▒▒│
  │ ═══│       └──┘
  └────┘
```

**Specs:**
- **サイズ**: 25 x 30 x 9mm (500円玉 x 2枚分)
- **重量**: ~5g
- **マイク**: INMP441 x1 (モノラル)
- **スピーカー**: なし (BLE → イヤホンで応答)
- **バッテリー**: 200mAh (録音のみ9時間)
- **装着**: ペンダント / マグネットクリップ / ピンバッジ
- **ユースケース**: 常時録音 → スマホ/イヤホンで応答受信

**特徴**: マイク+ESP32+バッテリーだけ。音声出力はBLE経由。最小最軽量。

## 2. Koe BAND — Xiaomi Smart Band風ウェアラブル

```
        ╭━━━━━━━━━━━━━━━╮
   ═════╡  ○ ○    [●LED] ╞═════
        │  mic    ▃▃▃▃▃  │
        │        speaker  │
        │    ── USB-C ──  │
        ╰━━━━━━━━━━━━━━━╯

  装着イメージ (手首)
         ╱━━━━━━━╲
    ━━━━╡ ○○ ● ▃▃╞━━━━
        │         │
         ╲━━━━━━━╱
           手首
```

**Specs:**
- **サイズ**: 42 x 18 x 11mm (本体) + シリコンバンド
- **重量**: ~15g (本体) + 10g (バンド) = 25g
- **マイク**: INMP441 x2 (ステレオ、ビームフォーミング)
- **スピーカー**: 10x8mm 楕円マイクロスピーカー (手首で聞ける)
- **バッテリー**: 120mAh (薄型カーブド)
- **追加センサー** (optional):
  - 心拍: MAX30102 (光学式)
  - 加速度: LIS3DH (ジェスチャー/歩数)
- **装着**: リストバンド (Xiaomi Band 8風の着脱式)
- **ユースケース**: 手首タップで会話、フィットネス連携

**デザインコンセプト:**
```
  ┌─ 参考: Xiaomi Smart Band 8 ─┐
  │ • カプセル型、端が丸い         │
  │ • マット+光沢のツートン         │
  │ • バンドは工具不要のクリック式   │
  │ • 画面なし → LEDバーで表現      │
  └────────────────────────────┘

  Koe BAND 独自:
  • 画面の代わりにLEDストリップ (WS2812B x 5)
    → 音声波形/音量/状態をアニメーション表示
  • サイドにタッチセンサー (ESP32 capacitive touch)
    → ダブルタップで録音、長押しでミュート
  • 底面にマイク穴 (肌側ではなく外側)
  • マグネット充電 (Pogoピン)
```

**カラー展開:**
- Obsidian Black (マットブラック + クローム)
- Lunar White (ホワイト + シルバー)
- Forest Green (ダークグリーン + ゴールド)

## 3. Koe STUDIO — 高音質スタンドアロン

```
      ╭━━━━━━━━━━━━━━━━━━╮
      │    ○        ○     │  ← デュアルマイク
      │                   │     (Wide spacing)
      │   ┌──────────┐    │
      │   │ ESP32-S3 │    │
      │   └──────────┘    │
      │                   │
      │  ╔══════════════╗ │  ← フルレンジスピーカー
      │  ║  SPEAKER     ║ │     28mm ドライバー
      │  ║  28mm        ║ │
      │  ╚══════════════╝ │
      │                   │
      │  [●] LED  [BTN]  │
      │     ─ USB-C ─    │
      ╰━━━━━━━━━━━━━━━━━━╯
          50 x 70 x 20mm

  側面
  ┌────────┐
  │▒▒▒▒▒▒▒▒│  ← 20mm (バッテリー+スピーカー)
  └────────┘
```

**Specs:**
- **サイズ**: 50 x 70 x 20mm (クレジットカードサイズ)
- **重量**: ~65g
- **マイク**: INMP441 x2 (Wide spacing = 40mm → 良好なビームフォーミング)
- **スピーカー**: 28mm フルレンジ 4ohm 3W
- **アンプ**: MAX98357A (3.2W → 十分な音圧)
- **バッテリー**: 2000mAh (最大24時間録音)
- **追加**: パッシブラジエーター (低音増強)
- **装着**: クリップ / スタンド / ストラップ
- **ユースケース**: 会議/デスク/アウトドア、スピーカーフォン品質

**音質向上のポイント:**
```
  1. スピーカー: 28mm フルレンジ (vs 15mm)
     → 200Hz〜20kHz カバー
     → 音量: 85dB @ 1m

  2. エンクロージャー: 密閉 + パッシブラジエーター
     ┌──────────┐
     │ ▓▓▓▓▓▓▓▓ │ ← Active driver (front)
     │ ┌──────┐ │
     │ │ AIR  │ │ ← Sealed chamber
     │ └──────┘ │
     │ ░░░░░░░░ │ ← Passive radiator (back)
     └──────────┘
     → 低音が劇的に改善 (60Hz〜)

  3. DSP: ESP32-S3のDSP機能
     → EQ調整、ノイズキャンセル、エコーキャンセル
     → AEC (Acoustic Echo Cancellation) で
       スピーカー出力がマイクに回り込むのを防止
```

## 共通設計原則

### 素材
- **本体**: アルミ (CNC) or PC+ABS (射出成形)
- **仕上げ**: サンドブラスト + アノダイズ (Xiaomi風)
- **ボタン**: セラミック or ステンレス
- **LED窓**: ポリカーボネート (透過光)

### Xiaomi Band 8 に学ぶデザイン要素
1. **カプセル形状** — 角を完全に丸める (R=短辺の半分)
2. **ツートン素材** — マットボディ + 光沢アクセント
3. **ミニマル** — ボタン1つ、LED1つ、ロゴなし
4. **触覚** — 側面のわずかなカーブ、持つと手に馴染む
5. **バンド着脱** — ワンタッチスライド (工具不要)
