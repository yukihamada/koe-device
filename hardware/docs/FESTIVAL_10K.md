## 機材リスト

### SUBウーファー (低音)

| 配置 | 台数 | 理由 |
|------|------|------|
| ステージ下 (グランドスタック) | 6 | メイン低音、カーディオイドアレイ (前4+後2逆位相) |
| ディレイタワーA (50m地点) | 2 | 中盤の低音補強 |
| ディレイタワーB (100m地点) | 2 | 後方の低音補強 |
| **合計** | **10** | |

なぜカーディオイド: 前方4台 + 後方2台(逆位相)でステージ裏への低音漏れを-15dB抑制。近隣への騒音対策。

### FILLスピーカー (中高域)

| 配置 | 台数 | 理由 |
|------|------|------|
| ステージ左右 (メインPA) | 6 | 3台x2スタック。前方50mをカバー |
| フロントフィル | 2 | VIPエリア前方の補強 |
| ディレイタワーA (50m) | 4 | 2台x2スタック。50-100mをカバー |
| ディレイタワーB (100m) | 4 | 2台x2スタック。100-150mをカバー |
| サイドフィル (オプション) | 2 | 横方向のカバレッジ補強 |
| **合計** | **18** | |

### COIN (観客デバイス)

| エリア | 人数 | COIN | 理由 |
|--------|------|------|------|
| VIP | 2,000 | 1,500 | 配布率75%。イヤホン+LED同期 |
| メイン | 5,000 | 2,000 | 配布率40%。近接補強+LED |
| 後方 | 3,000 | 500 | 配布率17%。距離補正の効果が最大 |
| **合計** | 10,000 | **4,000** | |

なぜ全員ではないか: 後方はFILL+SUBのディレイタワーで十分カバー。COINは「あると良い」であって必須ではない。VIPは全員配布でプレミアム体験。

### STAGE (制御)

| 台数 | 役割 |
|------|------|
| 1 | メインFOH (Front of House)。全機材制御 |
| 1 | 予備。メインがダウンした場合に自動切替 |
| **合計** | **2** |

### WiFiインフラ

| 機材 | 台数 | 理由 |
|------|------|------|
| 業務用WiFi 6E AP (Ubiquiti U6 Enterprise等) | 8 | 4000台のCOINを捌く。1台あたり500台 |
| PoEスイッチ (24ポート) | 1 | AP+STAGE+FILLの有線バックボーン |
| Cat6ケーブル (100m) | 8本 | AP配置用 |

なぜ業務用AP: ESP32は2.4GHz WiFi 4。1台のAPに500台接続は業務用でないと不安定。Ubiquiti U6は1024台同時接続対応。

|------|---------|--------|
| SUB x10 | 10kW (ピーク) | 100Vx3回路 |
| FILL x18 | 4.5kW (ピーク) | 100Vx2回路 |
| AP + STAGE + PoE | 0.5kW | 100Vx1回路 |
| **合計** | **~15kW** | **6回路** |

L-Acoustics同等構成: 50-80kW。**1/4の電力。**

## リスク管理

| リスク | 対策 |
|--------|------|
| WiFi飽和 (4000台) | 8x業務用AP + 2.4/5GHz分離 + COIN接続数制限 |
| GPS失測 (屋内/悪天候) | STAGE間のPTPフォールバック + TCXO holdover |
| SUB故障 | 6台中1台故障でも5台で運用可能 (音圧-2dB) |
| FILL故障 | ディレイタワー間の相互バックアップ |
| COIN電池切れ | 200mAh LiPo → ~5時間。替えのCOIN500台を用意 |
| 雨 | SUB/FILL: IP55。COIN: シリコンカバー付き |
| 盗難 (COIN) | $24原価なので諦める。デポジット制にするのも可 |
