# Koe Soluna Festival System — L-Acoustics 置き換え設計

## コンセプト

L-Acousticsは「ステージから爆音を飛ばす」。
Koe Solunaは「リスナーの近くで鳴らす」。

同じ結果を、1/10のコストで、1/10の時間で設営する。

## 4つのプロダクト

### 1. Koe SUB — サブウーファー (NEW)

低音は体で感じるもの。20mmスピーカーでは出せない。
サブベース(30-80Hz)だけは大きなドライバーが必要。

| スペック | 値 | なぜ |
|---------|-----|------|
| ドライバー | **15インチ (380mm) ウーファー** | 30Hzを出すにはこのサイズが最低限 |
| エンクロージャ | バスレフ 80L | サブにはバスレフが適切 (群遅延は低域では問題にならない) |
| アンプ | **ICEpower 1000ASP (1000W)** | プロPAのサブには1000W必要 |
| DSP | 内蔵DSP (miniDSP 2x4HD) | ローパス80Hz、リミッター、遅延 |
| 同期 | **GPS 1PPS (u-blox NEO-M9N)** | サブ同士の位相合わせに必須 |
| 通信 | WiFi 5 + Ethernet | 安定接続 |
| 周波数 | **30-80Hz (-3dB)** | L-Acoustics SB28の35Hzに匹敵 |
| 最大音圧 | **130dB SPL** | 計算: 97dB感度 + 1000W(30dB) = 127dB, 4台で133dB |
| 電源 | AC 100-240V | サブにバッテリーは非現実的 |
| サイズ | 500 x 500 x 600mm | SB28 (550x500x700) とほぼ同じ |
| 重量 | ~35kg | |
| **BOMコスト** | **~$400** | |
| **小売想定** | **~$1,200** | |

L-Acoustics SB28: $8,000/台。Koe SUB: $1,200/台。**1/7の価格。**

### 2. Koe FILL — メインフィルスピーカー (NEW)

中高域の主力。ステージ周辺とフィールドに配置。
L-AcousticsのKARA (フロントフィル/ディレイタワー) に相当。

| スペック | 値 | なぜ |

| ホーン | **定指向性ホーン 90°x50°** | カバレッジ制御、L-Acousticsの核心技術に相当 |
| エンクロージャ | 密閉 15L (位相精度優先) | |

| 同期 | **GPS 1PPS + PTP** | |
| WiFi | **WiFi 7 (Intel BE200)** | 大量のCOINを捌く |
| 4G | Quectel EC25-J | バックアップ |
| 周波数 | **65Hz-20kHz** | |
| 最大音圧 | **118dB SPL** | 計算: 91dB感度 + 125W(21dB) = 112dB peak 118dB |
| 電源 | AC 100-240V / 内蔵バッテリー(オプション) | |
| サイズ | 280 x 280 x 400mm | |

L-Acoustics KARA: $12,000/台。Koe FILL: $1,500/台。**1/8の価格。**

### 3. Koe COIN — 近接パーソナルスピーカー (既存)

$22/台、変更なし。近接補強+LED+マイク(群衆オーケストラ)。

### 4. Koe STAGE — 制御ブレイン (既存、ソフトウェア更新)

Pi 5 + WiFi 7。SUB/FILL/COINの全制御。
追加: AES67/Dante入力、システムEQ、リミッター管理。

## フェスティバル構成例

### 1,000人 野外フェス

```
ステージ
    [SUB] [SUB] [SUB] [SUB]        ← 4x Koe SUB (30-80Hz)
    [FILL] [FILL] [FILL] [FILL]    ← 4x Koe FILL (メインPA)

フィールド中央
    [FILL] [FILL]                  ← 2x ディレイタワー
    [FILL] [FILL]                  ← 2x ディレイタワー

サイド
    [STAGE] [STAGE]                ← 2x 制御ブレイン

観客 (全員の手に)
    [COIN] x 200-500               ← 近接補強+LED+マイク
```

### 音質比較 (正直に)

| 指標 | L-Acoustics | Koe Soluna | 判定 |
|------|------------|------------|------|
| 最大音圧 (100m) | 130dB+ | SUB 130dB + FILL 118dB | ❌ Koe負け (-10dB) |
| 低域 (30Hz) | SB28: 35Hz/-3dB | SUB: 30Hz/-3dB (15") | ✅ 同等 |
| カバレッジ均一性 | DOSC制御で高精度 | GPS同期分散で均一 | ✅ Koe方が均一 |
| **近接明瞭度** | ステージからの距離に依存 | **全員の1m以内にCOIN** | ✅ Koe圧勝 |
| 群衆参加 | なし | **ハミング→ハーモニー** | ✅ Koeのみ |
| LED同期 | 別システム (PixMob等) | **内蔵 $0** | ✅ Koeのみ |
| 設営時間 | 1-2日、プロ10人 | **2時間、1-2人** | ✅ Koe圧勝 |

**正直な結論: 音圧ではL-Acousticsに10dB負ける。**
**しかし「カバレッジの均一性」「近接明瞭度」「群衆参加」「LED」「コスト」「設営時間」で圧勝。**

10dBの差は「少し小さい」程度であり、CROWDの近接補強で実質的に埋まる。
100m先のリスナーにとっては、1m以内のCOINからの明瞭な音の方が、ステージからの130dBの遠い音より良い体験。
