なぜv2かWhy v2

BLE LE Audioがすべてを変えるBLE LE Audio changes everything

Bluetooth LE Audio（Auracast）は、1台の送信機から無限台の受信機へ、ペアリングなしでブロードキャストできる新規格。WiFiが不要になり、消費電力はさらに1/5に。LC3コーデックで音質も2倍になります。Bluetooth LE Audio (Auracast) is a new standard that broadcasts from one transmitter to unlimited receivers with zero pairing. Wi-Fi is no longer needed, power consumption drops to 1/5, and the LC3 codec delivers 2× the audio quality.

📡
Auracast — ペアリング不要Auracast — no pairing
BLE LE Audioのブロードキャストモード。受信機は電源を入れるだけ。台数制限なし。BLE LE Audio's broadcast mode. Receivers just power on. No limit on the number of devices.
