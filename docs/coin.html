特徴Features

声を、インフラにするMake voice the infrastructure

1台は記憶、100台はオーケストラ。Koe COINはESP32-S3 + ES8388コーデックで、マイク→ADPCM圧縮→UDPマルチキャストの送受信を同時に行います。One device remembers. A hundred become an orchestra. Koe COIN uses ESP32-S3 + ES8388 codec to simultaneously capture audio, compress with ADPCM, and transmit/receive over UDP multicast.

🎙
双方向リアルタイム音声Bi-directional real-time audio
マイク入力とスピーカー出力を同時処理。全デバイスが同期してオーケストラになる。Captures mic input and plays speaker output simultaneously — every device syncs into one orchestra.
⚡
8ms超低遅延ultra-low latency
32サンプルパケット + WiFi QoS (DSCP EF) + DMAチューニングで22ms → 8msを実現。32-sample packets + Wi-Fi QoS (DSCP EF) + DMA tuning cuts latency from 22ms to 8ms.
📡
WiFi UDP マルチキャストWi-Fi UDP multicast
239.42.42.1:4242 Solunaプロトコル。ルーター1台で256台同時接続。Soluna protocol on 239.42.42.1:4242. 256 concurrent devices on a single router.
🔄
OTA ファームウェア更新OTA firmware update
koe.live経由でフィールドアップデート。WiFi設定・デバイスIDは消えない。Field updates via koe.live. Wi-Fi settings and device IDs are preserved.
🎛
ボタン操作Button controls
録音ON/OFF、ピッチシフト（±5/±12半音）、ボリューム調整をワンボタンで。Record on/off, pitch shift (±5/±12 semitones), and volume — all single-button.
