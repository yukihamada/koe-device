機能Features

キーボードより速いFaster than typing

ローカルで動作する音声認識。キーを押している間だけ録音 → 離したらカーソル位置に転写される。クラウド送信なし。Local speech recognition. Record while the key is held, transcribe at cursor release. No cloud upload.

⚡
Shift長押しで録音Hold Shift to record
右Shift長押し中だけマイクが動き、離した瞬間に転写される。The mic is only active while Right Shift is held. Release to transcribe.
🔒
ローカル処理Local processing
ON-DEVICE SPEECHで処理。マイク音声は外部に送信されない。Processed on-device via SFSpeechRecognizer. Audio never leaves your Mac.
🌐
日英自動判定JA/EN auto-detect
日本語/英語を話すたびに自動で認識言語を切替。設定不要。Auto-switches between Japanese and English based on what you speak.
🍏
Apple Silicon最適化Apple Silicon native
M1/M2/M3/M4ネイティブ。Intelでも動作。軽量でバッテリー友達。Native on M1/M2/M3/M4 Macs. Runs on Intel too. Lightweight, battery-friendly.
🎯
どのアプリでもWorks everywhere
キーボード入力を受け付けるアプリなら全て対応。ブラウザ、エディタ、メーラー、チャット。Works with any app that accepts keyboard input — browsers, editors, mail, chat.

Koe Device との関係Relationship to Koe Device

別プロダクト、同じ思想Different product, same philosophy

Koe 音声入力はソフトウェア単独で動作するmacOSアプリ。Koe DeviceのESP32ハードウェアとは独立していますが、「声を、インフラにする」という同じ思想で作られています。Koe Voice Input is a standalone macOS app. It operates independently from the Koe Device ESP32 hardware, but shares the same philosophy: make voice the infrastructure.
