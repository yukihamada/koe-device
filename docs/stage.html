Koe STAGEとはAbout Koe STAGE

ミキシングを民主化するDemocratize mixing

Pi CM5 + Rust製ハブソフトウェアで、32チャンネルの音声をリアルタイムミキシング。フェス、コンサート、スタジオ、配信スタジオに。Soluna OS ダッシュボードで全デバイスを可視化。A Pi CM5 + Rust-based hub that mixes 32 channels in real time. For festivals, concerts, studios, and streaming rooms. Visualize every device via the Soluna OS dashboard.

🎛
32ch リアルタイムミキシング32ch real-time mixing
Koe COINからのUDP音声を32チャンネル同時処理。EQ・リバーブ・コンプレッサー内蔵。Processes 32 UDP channels from Koe COIN simultaneously. Built-in EQ, reverb, and compressor.
📺
SRT / RTMP 配信SRT / RTMP streaming
YouTube・Twitch・OBS連携。ミックス済み音声をリアルタイム配信。Integrates with YouTube, Twitch, and OBS. Streams mixed audio in real time.
📊
Soluna OS ダッシュボードSoluna OS dashboard
256台のKoeデバイスの音量・ステータス・位置情報をWebダッシュボードで管理。Manage volume, status, and location of up to 256 Koe devices through a web dashboard. → デモを見る→ Live demo

ゼロクラッシュ設計。tokio非同期ランタイムで低レイテンシー処理。Zero-crash design. Low-latency processing on the tokio async runtime.
