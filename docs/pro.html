Koe Pro — The Transmitter — トランスミッター

System Architecture システムアーキテクチャ

BLE LE Audio carries the signal. UWB provides sub-microsecond clock sync. BLE LE Audioで信号伝送。UWBでサブマイクロ秒のクロック同期。

🎸

Live Band Performanceライブバンド演奏

Wireless stage without the cable mess. Each musician wears a Koe Pro. Hub mixes and sends to PA.ケーブルなしのワイヤレスステージ。各ミュージシャンがKoe Proを装着。HubでミックスしてPAへ出力。

🎧

Street Music / Buskingストリートミュージック

Portable setup. Pro + Hub + powered speaker. Set up in minutes, no cable runs.ポータブルセットアップ。Pro + Hub + パワードスピーカー。数分でセッティング完了。

🎶
